
## 1. Product Overview
网页版斗地主纸牌游戏，支持玩家与两个AI电脑对战，包含完整的叫地主、抢地主和出牌流程。
- 核心玩法：经典斗地主规则，三个玩家（1个用户+2个AI）
- 目标用户：喜欢纸牌游戏的休闲玩家，提供娱乐体验
- 市场价值：快速上手、无需安装的网页游戏，适合碎片化时间娱乐

## 2. Core Features

### 2.1 User Roles (if applicable)
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| 玩家 | 无需注册 | 进行游戏、叫地主、出牌 |

### 2.2 Feature Module
1. **游戏主页面**：玩家区域、AI区域、牌堆、游戏控制
2. **金币系统**：初始1000金币，用于叫地主和抢地主
3. **叫地主阶段**：底分1金币，最高3金币
4. **出牌阶段**：支持各种斗地主牌型
5. **手牌数显示**：实时展示各个玩家剩余手牌数
6. **胜负判定**：计算得分，更新金币
7. **游戏结束**：金币输光时显示游戏结束

| 游戏主页面 | AI玩家区 | 显示两个AI玩家的牌背 |
| 游戏主页面 | 地主牌区 | 显示地主的三张底牌 |
| 游戏主页面 | 金币显示 | 显示当前玩家金币数 |

| 游戏主页面 | 操作按钮 | 叫地主、抢地主、不叫、出牌、不出等按钮 |

## 3. Core Process
游戏流程：发牌 → 叫地主 → 抢地主 → 确定地主 → 地主拿底牌 → 地主先出牌 → 轮流出牌 → 一方出完牌 → 结算金币 → 开始新游戏或游戏结束
```mermaid
flowchart TD
    A[发牌] --&gt; B[叫地主阶段]
    B --&gt; C{有人叫地主?}
    C --&gt;|是| D[抢地主阶段]
    C --&gt;|否| E[重新发牌]
    D --&gt; F[确定地主]
    F --&gt; G[地主拿底牌]
    G --&gt; H[出牌阶段]
    H --&gt; I{一方出完牌?}
    I --&gt;|否| H
    I --&gt;|是| J[结算金币]
    J --&gt; K{金币输光?}
    K --&gt;|否| A
    K --&gt;|是| L[游戏结束]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深绿色 (#0d5016) 和金色 (#f59e0b)，营造扑克牌桌的氛围
- 按钮风格：圆角按钮，带渐变和阴影
- 字体：使用 'Noto Sans SC' 中文字体，搭配 'Poppins' 英文字体
- 布局风格：卡片式布局，深色背景，金色点缀
- 图标风格：使用 lucide-react 图标库

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 游戏主页面 | 游戏桌面 | 深绿色背景，金色边框，模拟真实扑克牌桌 |
| 游戏主页面 | 纸牌 | 标准纸牌样式，支持展开和选中效果 |
| 游戏主页面 | 按钮区 | 金色渐变按钮，带悬停和点击动画 |
| 游戏主页面 | 信息栏 | 金色文字，显示金币、底分等信息 |
| 游戏主页面 | 手牌数显示 | 每个玩家区域显示剩余手牌数量，使用数字标识 |

### 4.3 Responsiveness
- 桌面端优先设计
- 支持响应式布局，适配平板设备
- 触摸优化，支持移动设备操作

### 4.4 3D Scene Guidance (if applicable)
本项目不包含3D场景
